# Product Definition: kdex-entitlements

## Core Concept
A simple, robust entitlements checking library written in Go. It is designed to provide efficient and reliable entitlement validation, enabling applications to easily determine user or service access rights.

## Goals
*   Provide a simple API for checking user and system entitlements.
*   Ensure high performance and reliability for authorization checks.
*   Maintain a lightweight footprint with minimal external dependencies.

## Target Audience
*   Developers building Go applications that require authorization and entitlement management.
